# ChatGPT/Codex Prompts Library

This directory contains reusable prompts for ChatGPT and Codex automation aligned with housing policy research goals.

## Purpose

- Standardize AI-generated content
- Track prompt evolution
- Enable reproducibility
- Facilitate review and quality control

## Structure

Organize prompts by:
- `policy_analysis/` - Prompts for policy research and analysis
- `data_processing/` - Prompts for data cleaning and transformation
- `documentation/` - Prompts for generating documentation
- `literature_review/` - Prompts for research summaries

## Format

Each prompt file should include:
```markdown
# Prompt: [Title]

**Purpose:** [Description]
**Model:** [e.g., gpt-4-turbo-preview]
**Date Created:** [YYYY-MM-DD]
**Author:** [Name]

## Prompt Text
[The actual prompt]

## Expected Output
[Description of expected format/content]

## Evaluation Criteria
[How to assess quality]
```

## Usage

Reference prompts in scripts and workflows to maintain consistency across the project.
